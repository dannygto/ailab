# AILAB 开发指南（标准化）

## 环境要求
- Node.js >= 16.x
- npm >= 8.x
- Python >= 3.8（AI服务）
- MongoDB >= 4.4
- Redis >= 6.0（可选）
- Docker & Docker Compose（推荐）
- 推荐IDE：VS Code

## 依赖安装
```bash
git clone [项目地址]
cd ailab
powershell -ExecutionPolicy Bypass -File "scripts/build.ps1" -install
```

## 启动流程
- 推荐使用统一启动脚本：
```powershell
powershell -ExecutionPolicy Bypass -File "scripts/start-platform.ps1" -all -dev
```
- 也可分别启动前端、后端、AI服务

## 代码规范
- TypeScript全栈，ESLint+Prettier自动格式化
- 组件/模块命名采用PascalCase
- 严禁any类型，接口/类型定义齐全

## 分支与协作
- main分支：生产稳定
- develop分支：开发集成
- feature/、hotfix/分支：功能/修复
- 所有合并需Code Review

## 自动化测试
- 前端：Jest + React Testing Library
- 后端：Jest + Supertest
- AI服务：单元测试/Mock
- 所有PR需通过测试

## CI/CD
- 支持GitHub Actions/自建CI，自动测试、构建、部署

## 调试与工具
- Chrome DevTools、VS Code调试、console.log/winston日志
- 推荐使用Mock/测试数据

## 国际化预留
- 代码与文档结构支持多语言扩展
