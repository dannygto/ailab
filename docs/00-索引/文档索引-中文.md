## 端口配置

在所有文档和配置文件中统一使用以下端口：

| 服务        | 端口  | 说明                     |
|-----------|-----|------------------------|
| 前端        | 3000 | React应用               |
| 后端        | 3001 | Node.js API服务         |
| 监控（Grafana）| 3002 | 系统监控仪表盘              |
| MongoDB   | 27017 | 数据库（仅内部访问）          |
